| الرمز    | الوصف                                        | مثال        | النتيجة                                       |        |                      |
| -------- | -------------------------------------------- | ----------- | --------------------------------------------- | ------ | -------------------- |
| `.`      | أي حرف أو رقم أو رمز (باستثناء السطر الجديد) | `/a.b/`     | يطابق "acb"، "a1b"، "a-b"                     |        |                      |
| `^`      | بداية النص                                   | `/^Hello/`  | يطابق "Hello world" لكنه لا يطابق "Say Hello" |        |                      |
| `$`      | نهاية النص                                   | `/world$/`  | يطابق "Hello world" لكنه لا يطابق "worldwide" |        |                      |
| `\d`     | أي رقم من 0 إلى 9                            | `/\d/`      | يطابق "1" في "abc1"                           |        |                      |
| `\D`     | أي حرف غير رقم                               | `/\D/`      | يطابق "a" في "a1"                             |        |                      |
| `\w`     | أي حرف أو رقم أو underscore `_`              | `/\w/`      | يطابق "a" في "a1\_"                           |        |                      |
| `\W`     | أي حرف غير حرف أو رقم أو underscore          | `/\W/`      | يطابق "-" في "a-b"                            |        |                      |
| `\s`     | أي فراغ (space, tab, newline)                | `/\s/`      | يطابق " " في "a b"                            |        |                      |
| `\S`     | أي حرف غير فراغ                              | `/\S/`      | يطابق "a" في "a b"                            |        |                      |
| `[abc]`  | أي حرف من المجموعة                           | `/[abc]/`   | يطابق "a" أو "b" أو "c"                       |        |                      |
| `[^abc]` | أي حرف ليس من المجموعة                       | `/[^abc]/`  | يطابق أي حرف غير a,b,c                        |        |                      |
| `[a-z]`  | أي حرف صغير                                  | `/[a-z]/`   | يطابق "b" في "abc"                            |        |                      |
| `[A-Z]`  | أي حرف كبير                                  | `/[A-Z]/`   | يطابق "C" في "ABC"                            |        |                      |
| `[0-9]`  | أي رقم                                       | `/[0-9]/`   | يطابق "5" في "a5c"                            |        |                      |
| `+`      | واحد أو أكثر من العنصر السابق                | `/a+/`      | يطابق "a", "aa", "aaa"                        |        |                      |
| `*`      | صفر أو أكثر من العنصر السابق                 | `/a*/`      | يطابق "", "a", "aa"                           |        |                      |
| `?`      | صفر أو واحد من العنصر السابق                 | `/a?/`      | يطابق "", "a"                                 |        |                      |
| `{n}`    | يطابق العنصر n مرة بالضبط                    | `/a{3}/`    | يطابق "aaa" فقط                               |        |                      |
| `{n,}`   | يطابق العنصر n مرة أو أكثر                   | `/a{2,}/`   | يطابق "aa", "aaa" ...                         |        |                      |
| `{n,m}`  | يطابق العنصر بين n و m مرات                  | `/a{2,4}/`  | يطابق "aa", "aaa", "aaaa"                     |        |                      |
| `()`     | تجميع العناصر                                | `/(ab)+/`   | يطابق "ab", "abab"                            |        |                      |
| \`       | \`                                           | OR (أو)     | \`/cat                                        | dog/\` | يطابق "cat" أو "dog" |
| `\b`     | حد الكلمة                                    | `/\bcat\b/` | يطابق "cat" ككلمة منفصلة فقط                  |        |                      |
| `\B`     | ليس حد الكلمة                                | `/\Bcat\B/` | يطابق "scatter" (داخل كلمة)                   |        |                      |
| `(?i)`   | تجاهل حالة الأحرف (case-insensitive)         | `/(?i)abc/` | يطابق "ABC" أو "abc"                          |        |                      |
